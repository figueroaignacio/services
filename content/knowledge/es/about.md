# Sobre Ignacio Figueroa
Ignacio Figueroa es un desarrollador web freelance ubicado en Argentina. Desarrolla sitios web profesionales de alta calidad, landing pages, aplicaciones web a medida e integraciones con Inteligencia Artificial (chatbots y automatizaciones) para clientes de todo el mundo.

## ¿Por qué trabajar con Ignacio?
- **Sin Plantillas**: Cada sitio está diseñado y redactado a medida desde cero para convertir visitantes.
- **Contacto Directo**: Hablás directamente con Ignacio, sin intermediarios, ejecutivos de cuenta ni burocracia de agencias.
- **Servicio Llave en Mano**: Nos encargamos del diseño, desarrollo, redacción de textos, hosting, dominio, seguridad activa y actualizaciones mensuales.

## Stack Tecnológico
- **Frontend**: Next.js, React, Tailwind CSS, TypeScript, animaciones CSS modernas
- **Backend**: Node.js, Express, Serverless Functions, Resend API
- **Bases de Datos**: PostgreSQL, Redis, Supabase
- **Integraciones**: Pagos con Stripe y Mercado Pago, embudos de CRM (HubSpot, Salesforce), API de WhatsApp, Chatbots con IA (Vercel AI SDK, Gemini, OpenAI)
- **Deployment**: Vercel, Netlify, Cloudflare
